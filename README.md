# 演示 Demo

## 使用方式

```
# 1.安装npm包
npm install

# 2.运行
npm run dev

# 3.打开微信开发者工具，新建项目，选择dist目录，appid用测试的就行，项目名自己起
```

# 说明

示例演示了 Button 组件和 Notify 组件的使用方式。请对照[Vant 文档](https://youzan.github.io/vant-weapp/#/notify)进行理解。
